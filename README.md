# Laptop-Specifications-Data-Analysis-Visualization-Python-Power-BI-
Data analysis project exploring laptop specifications such as price, CPU, RAM, GPU, display, and battery using Python and Power BI for insights and visualization.

## 📊 Project Overview

This project analyzes a dataset of laptop specifications to understand trends in pricing, hardware configuration, and performance features. The analysis is performed using **Python for data processing and exploratory analysis** and **Power BI for interactive visualization**.

The goal of this project is to extract meaningful insights from laptop specifications such as CPU type, RAM size, storage, GPU, display features, and battery capacity.

---

## 🎯 Objectives

The main objectives of this project are:

* Explore laptop hardware specifications
* Analyze pricing patterns across different laptop configurations
* Understand relationships between hardware components and price
* Identify trends in CPU, RAM, GPU, and storage
* Visualize insights using Power BI dashboards

---

## 📂 Dataset Features

The dataset contains various laptop attributes including:

* Laptop Name
* Price (EUR)
* Display Size
* Display Resolution
* Aspect Ratio
* Weight
* Dimensions
* Operating System
* CPU Processor
* RAM Memory
* GPU (Integrated / Dedicated)
* Storage Type
* Battery Life
* PSU Watts
* Audio System
* Number of Speakers
* Touchscreen Support

---

## 📈 Key Insights

Example insights derived from the analysis:

* High RAM and dedicated GPU laptops tend to have higher prices.
* SSD storage is the most common storage type in modern laptops.
* Lightweight laptops generally have smaller display sizes.
* Intel processors dominate most laptop models in the dataset.
